# 인텔즈 / 유북 — 이전 경력

[← 포털로 돌아가기](../README.md) | [English](../en/intelz.md)

## 개요

[인텔즈 (Intelz)](https://youbook.biz)에서 **앱 개발자 & 데이터 매니저** — AI 기반 미니북 지식 공유 플랫폼. "동영상은 YouTube, 미니북은 YouBook."

**Organization:** [plootokr](https://github.com/plootokr)
**기간:** 2022.02 — 2025.08 (약 3.5년, 대학 재학 중)
**학력:** 서울대학교 건설환경공학부 (2025.07 졸업)
**커밋:** 6개 레포지토리에 368건

## 프로덕트

유북(YouBook)은 사용자가 10페이지 이내의 "미니북"을 생성, 발견, 리믹스하는 지식 공유 플랫폼. AI 기반 콘텐츠 공동 제작(GPT-4), 전문가 화상 멘토링, 13개 이상 언어 자동 번역으로 190개국 이상에 서비스하며, 저자 70% 수익 분배 모델 운영.

플랫폼은 세 번의 브랜드 전환을 거침: Plooto → Intelz → YouBook.

## 팀

4인 팀: 백엔드 개발자, 웹 개발자, **앱 개발자 (본인)**, 디자이너.

## 배운 것

이 역할은 코드만큼이나 비즈니스와 프로덕트에 관한 것이었습니다. 대학 재학 중 콘텐츠 플랫폼을 만들면서 사용자가 UX를 어떻게 인식하는지, 콘텐츠 구조가 어떻게 작동하는지, 프로덕트 결정이 엔지니어링 우선순위를 어떻게 형성하는지 배웠습니다. 모바일 앱, 데이터 파이프라인, 웹 통합이라는 기술적 작업은 수단이었고, 프로덕트 사고가 진짜 수확이었습니다.

## 구축한 것들

### 1. 모바일 앱 — 3세대 진화

3년간 3번의 리라이트를 통해 전체 모바일 앱 진화를 주도. React Native의 네이티브 브릿지 동작 원리(RN 0.73부터), 웹뷰와 네이티브 경험 간의 격차 관리, 모바일 플랫폼 제약이 프로덕트 결정에 미치는 영향을 이 과정에서 체득했습니다.

| 세대 | 레포 | 기간 | 커밋 | 아키텍처 |
|------|------|------|------|----------|
| 1세대 | service-app | 2022-02 → 2022-09 | 12 | Bare React Native, 웹뷰 전용 |
| 2세대 | intelzRN | 2022-07 → 2024-04 | 98 | 네이티브 React Native (v1.0→v1.6) |
| 3세대 | youbook-app | 2024-04 → 2025-08 | 119 | Expo managed workflow |

**각 세대가 추가한 것:**
- **1세대:** 웹 플랫폼용 기본 웹뷰 래퍼
- **2세대:** 네이티브 결제(iamport/PortOne), 푸시 알림(Firebase), CodePush OTA, Fastlane CI/CD, 카카오/Apple 소셜 로그인, 다국어 알림 구독
- **3세대:** Expo prebuild 마이그레이션, 최신 알림 처리(Notifee), 라이브러리 현대화

**앱스토어 & 플레이스토어 제출:** 전체 릴리스 파이프라인을 담당 — 앱 메타데이터, 스크린샷, 개인정보 보호 선언, 심사 가이드라인 준수, 버전 관리, 심사자 피드백 대응. iOS와 Android 양쪽의 초기 제출부터 지속적 업데이트까지 처리.

이 경험이 [부톡 앱 Expo 마이그레이션](projects/bootalk-app.md)의 직접적 기반 — 동일한 패턴을 두 번째에는 더 빠르게 실행.

### 2. 데이터 자동화 — 10배 속도 향상

**문제:** 콘텐츠 데이터를 수동으로 입력 — 느리고 오류가 잦은 프로세스로 상당한 팀 시간 소모.

**해결:** Selenium과 Python 크롤러를 활용한 자동 데이터 파이프라인 구축. 콘텐츠 데이터의 수집, 변환, 적재 자동화.

**임팩트:** 데이터 처리 시간 10배 단축. 이 작업이 부톡에서 CDO 역할의 기반 — Dagster 기반 ETL 파이프라인 구축 및 데이터팀 리드로 이어짐.

### 3. 웹-앱 통합 (service-web에 121건 커밋)

앱 개발자로서 웹 플랫폼과 네이티브 앱 간 브릿지를 담당:
- 앱 알림 뱃지 및 읽지 않은 메시지 카운트 동기화
- 인앱 결제 흐름 (iOS/Android)
- WebView ↔ 네이티브 브릿지 (Android 뒤로가기, 오디오 재생 시 화면 켜짐 유지)
- 소셜 로그인 통합 (카카오, Apple)
- 애널리틱스 통합 (GA4, Firebase)

### 4. 백엔드 — AI 챗봇 통합 (16건 커밋)

- 인텔즈봇(지식 챗봇)에 OpenAI GPT-4 통합, 실시간 스트리밍 응답
- Firebase 푸시 알림 인프라
- 네이티브 앱 인증 API
