## 프레임워크 & 라이브러리

### 프론트엔드
- **React / Next.js** — SSR, ISR, 동적 사이트맵, SEO/GEO용 구조화 데이터
- **React Native (Expo)** — OTA 업데이트, 딥링크, 네이티브 모듈
- **Zustand** — 상태 관리 (Recoil에서 마이그레이션)
- **styled-components** — CSS-in-JS (v5→v6 마이그레이션)
- **GraphQL** — 타입 안전한 클라이언트 통합

### 데이터 엔지니어링
- **Dagster** — 워크플로우 오케스트레이션, 에셋 기반 파이프라인
- **Pandas / Jupyter** — 데이터 분석, 변환
- **공공 API** — 실거래가, 분양 정보, 지역 코드

### 백엔드
- **Kotlin / Spring** — MSA 백엔드 서비스
- **AWS Lambda** — 서버리스 함수, 이벤트 트리거
- **REST API** — 설계, 버전 관리, 에러 핸들링

### 문서화
- **Mintlify** — 개발자 문서 플랫폼
- **MDX** — 인터랙티브 컴포넌트 포함 문서

### 모니터링 & 관측성
- **Sentry** — 에러 추적, 성능 모니터링, 릴리스 추적
- **Telegram Bot** — 파이프라인 모니터링, 데이터 변동 알림

### 인프라 & DevOps
- **AWS** — Lambda, Amplify, S3, CloudFront, SNS/SQS
- **GCP** — Cloud Run, Cloud SQL
- **Naver Cloud** — CDN+
- **GitHub Actions** — CI/CD 파이프라인
- **Expo EAS** — 모바일 앱 빌드 및 OTA 배포
- **Docker** — 컨테이너화

### AI 엔지니어링 (핵심 역량)

단순한 AI 활용이 아닌 — **프로덕션 워크플로우로서의 AI 오케스트레이션**.

- **Claude Code** — 주력 AI 코딩 에이전트 (2025년 8월 Cline에서 전환). 팀 전체 Max 20x 플랜 구매.
- **Tmux 멀티 에이전트 오케스트레이션** — 4-8개 동시 Claude Code 세션, 비중복 파일셋 운영
- **CLAUDE.md 아키텍처** — 레포별 지침 파일로 프로젝트 컨텍스트, 컨벤션, 제약조건을 AI 에이전트에 전달
- **야간 자율 실행** — 에이전트가 연속 처리하는 작업 큐, 리뷰 가능한 PR 생산
- **페이즈 기반 병렬 처리** — 대규모 캠페인을 독립 페이즈로 분해, 에이전트 세션에 분산
- **Claude GitHub Actions** — 6개 이상 레포지토리에 자동 PR 리뷰 및 코드 분석 워크플로우
- **oh-my-claudecode** — 복잡한 작업을 위한 고급 멀티 에이전트 오케스트레이션 프레임워크

**임팩트:** 13배 산출량 증가 (월 65건 → 838건). 단일 엔지니어가 팀 수준의 산출물 생산.

### AI/ML (프로덕트)
- **RAG** — 아파트 단지/매물 검색 증강 생성
- **OCR** — 이미지 및 테이블 추출
- **GPT 통합** — 스트리밍 SSE, 세무 상담 (세무GPT)
- **GEO** — AI 검색엔진 최적화 (ChatGPT, Perplexity, Gemini)

| **보안** | IDOR 수정, 개인정보보호법 준수, 서비스 인증 구현 |
