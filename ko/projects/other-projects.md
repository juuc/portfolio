## MSA 백엔드
**19건 커밋 | 10건 PR (10건 머지)**

Kotlin/Spring 멀티 모듈 MSA 백엔드 서비스.

**비즈니스 임팩트:**
- **배치 잡 최적화:** 일일 푸시 알림 배치 잡 최적화 — 커버링 인덱스로 쿼리 2,370배 향상, N+1 제거로 DB 라운드트립 98% 감소. 잡 실행 시간 8.4분 → 1분 미만. AI 자동화 EXPLAIN 포렌식 분석 도구를 구축하여 프로덕션 8개 쿼리를 분석하고 추가 최적화 대상 식별 — 플랫폼 전반의 비효율 쿼리를 반복적·자동으로 탐지 가능. ~1주 만에 완료.
- **IDOR 보안 수정:** 에이전트 서비스 전반에 `/me/` 패턴 엔드포인트 추가, IDOR(Insecure Direct Object Reference) 취약점 방지. 경로 기반 에이전트 ID 접근을 JWT 기반 인증으로 교체.
- **모놀리식 API 분해:** 대형 모놀리식 API를 5개 독립 RESTful 엔드포인트로 분리, 유지보수성 및 성능 개선.
- **데이터 서비스 수정:** 걷기 좋은 길 좌표 매핑, 거리 파라미터 타입 수정.

## 세무GPT — AI 세무 상담
**10건 커밋**

세무GPT 프론트엔드 — 파트너 세무법인을 위해 구축한 AI 세무 상담 서비스. 풀스택 개발이 필요한 주요 프로덕트 이니셔티브:

**구축한 것:**
- **결제 시스템:** PortOne 결제 게이트웨이 연동, free/standard/special 티어 과금
- **법령 데이터 파이프라인:** 세법(조문 단위), 판례, 집행기준을 Elasticsearch에 인덱싱. 법령-판례 교차참조 매핑 구현. 기본통칙, PDF 기반 집행기준 추가 수집
- **에이전틱 워크플로우:** 미니 모델 기반 쿼리 리라이팅 및 카테고리 분류. "특정 법의 특정 시기" API 호출 에이전틱 응답
- **평가:** RAGAS 프레임워크 기반 응답 품질 평가 환경 구축
- **채팅 UX:** 동적 라우트 `/chat/{id}` 대화 이력 조회, 응답별 피드백 폼, 법령/판례/상담 접기/펼치기, 카테고리 선택 토글
- **데이터 품질:** 판례 링크 오류 수정(재수집), 3개 이상 판례 제시 보장, 신법/구법 비교 표시 개선

**비즈니스 임팩트:** 2026년 1월 MVP 완성. 첫 클라이언트 미팅 완료. 파트너 피드백 기반 UX 및 데이터 품질 지속 개선 중.
